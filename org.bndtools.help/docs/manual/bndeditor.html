Bnd Editor

The bnd editor provides a number of tabs with graphic editors. The following tabs are available:

  • Contents – The content and resulting imports of the selected bundle
  • Description – Editor for the OSGi headers that describe a bundle
  • Build – The build path
  • Run – Create a launch configuration
  • Tests – Managing tests
  • Source – Source code editor for the bnd properties file
