export

Synopsis:

   export [options]  ...

Options:

  • [ -e --exclude <string;> ] Exclude files by pattern
  • [ -E --exporter <string>* ]
  • [ -o --output <string> ]
  • [ -p --project <string> ] Identify another project
  • [ -v --verbose ] prints more processing information
  • [ -w --workspace <string> ] Use the following workspace
