eclipse

Synopsis:

   eclipse [options]  ...

Options:

  • [ -d --dir <string> ] Path to the project
  • [ -w --workspace <string> ] Use the following workspace

Available sub-commands

  • sync - Synchronized the ./cnf/.settings directory to all the projects

sync

   sync
