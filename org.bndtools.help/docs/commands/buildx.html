buildx

Synopsis:

   buildx [options]  ...

Options:

  • [ -c --classpath <string>* ]
  • [ -e --eclipse ]
  • [ -f --force ]
  • [ -n --noeclipse ]
  • [ -o --output <string> ]
  • [ -p --pom ]
  • [ -s --sourcepath <string>* ]
  • [ -S --sources ]

Examples

bnd buildx -classpath bin -noeclipse -output test.jar xyz.bnd
