### Synopsis: 
	   release [options]  ...


#### Options: 
- `[ -p --project <string> ]` Path to project, default is current project
- `[ -r --repo <string> ]` Set the release repository
- `[ -t --test ]` Release with test build
- `[ -w --workspace ]` Release all bundles in in the workspace
